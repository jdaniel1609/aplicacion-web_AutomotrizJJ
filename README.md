# aplicacion-web_AutomotrizJJ

Las credenciales se configuran en Kubernetes Secrets

Kubernetes Secret "automotriz-jj-secrets"
├── AZURE_SQL_SERVER      → tu-servidor.database.windows.net
├── AZURE_SQL_DATABASE    → automotriz_jj_db
├── AZURE_SQL_USERNAME    → sqladmin
├── AZURE_SQL_PASSWORD    → Tu_Password!
└── SECRET_KEY            → clave_jwt_larga
